Ordinal Numbers in Portuguese

Learn how to express order and ranking in Portuguese

Basic Ordinal Numbers

Ordinal numbers in Portuguese are used to indicate order or position. They must agree in gender with the noun they modify.

Number Masculine Feminine English
1º primeiro primeira first
2º segundo segunda second
3º terceiro terceira third
4º quarto quarta fourth
5º quinto quinta fifth

Quick Tip

Unlike English, Portuguese ordinal numbers must agree in gender with the noun they modify. For example:

  • o primeiro dia (the first day) - masculine
  • a primeira noite (the first night) - feminine

Formation Pattern

Number Pattern Masculine Feminine
6º-9º Add -o/-a sexto, sétimo sexta, sétima
10º+ Add -ésimo/-ésima décimo, vigésimo décima, vigésima

Common Higher Numbers

  • 10th - décimo(a)
  • 20th - vigésimo(a)
  • 30th - trigésimo(a)
  • 40th - quadragésimo(a)
  • 50th - quinquagésimo(a)
  • 100th - centésimo(a)

Compound Ordinals

Examples of Compound Ordinals

  • 21st - vigésimo(a) primeiro(a)
  • 32nd - trigésimo(a) segundo(a)
  • 45th - quadragésimo(a) quinto(a)

Note: All parts of the compound ordinal must agree in gender with the noun.

Important Note

Unlike English, Portuguese ordinal numbers are generally only used up to 100th (centésimo). After that, it's more common to use cardinal numbers.

Common Uses

Frequent Applications

  • Floor numbers: o terceiro andar (the third floor)
  • Centuries: o século décimo nono (the nineteenth century)
  • Academic years: o primeiro ano (the first year)
  • Rankings: em segundo lugar (in second place)

Common Mistakes

1. Wrong Gender Agreement

a primeiro vez → a primeira vez

Remember to match the gender of the noun!

2. Compound Number Order

primeiro vigésimo andar → vigésimo primeiro andar

Larger number always comes first in compounds.

3. Abbreviation Format

1er, 2do → 1º, 2º

Portuguese uses º and ª for masculine and feminine abbreviations.

4. Using Cardinals Instead of Ordinals

o cinco andar → o quinto andar

Use ordinal numbers for order and ranking!

Abbreviations

Writing Ordinal Numbers

Portuguese uses special symbols for abbreviated ordinal numbers:

  • Masculine: º (1º, 2º, 3º)
  • Feminine: ª (1ª, 2ª, 3ª)
  • Plural adds s: 1ºs, 1ªs
